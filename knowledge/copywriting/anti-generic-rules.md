# Anti-Generic Rules — BabyMania Copywriting

These rules prevent AI-generated filler language from appearing in product copy.
Apply before finalizing any section output.

---

## Forbidden Phrases (English equivalents — apply to Hebrew translations too)

| Forbidden | Why it fails | Write instead |
|-----------|-------------|---------------|
| "high quality fabric" | applies to every product | name the specific quality: "בד רך שלא מתכווץ בכביסה" |
| "perfect for babies" | vague, unverifiable | describe the actual benefit: "נוח ללבישה יומיומית מגיל יומו" |
| "premium comfort" | marketing noise | describe the sensation: "בד קליל שמאפשר תנועה חופשית" |
| "great for everyday use" | says nothing | show the scenario: "מתאים לבית, לטיול ולביקורים" |
| "suitable for sensitive skin" | claim requires source | only write if fabric data confirms it |
| "carefully crafted" / "מעוצב בקפידה" | empty phrase | replace with the specific design detail |
| "selected with care" / "נבחר בקפידה" | means nothing | delete entirely |
| "the best" / "הכי טוב" | unprovable | delete — never use superlatives |
| "premium quality" / "איכות פרימיום" | vague | describe what makes it premium |
| "our products" | generic | refer to the specific item by name or type |

---

## The Concrete Test

Before publishing any sentence, ask:
**"Does this sentence apply to every baby garment in existence?"**

If yes → rewrite it with a specific, observable detail from the product data.
If no → it passes.

### Examples

**Fails the test (applies to everything):**
- "בד נוח ואיכותי לתינוקכם"
- "מוצר פרימיום שנבחר בקפידה"
- "מתאים לשימוש יומיומי"

**Passes the test (specific to this product):**
- "פתחים בתחתית — מאפשרים החלפת חיתול מהירה גם בלילה"
- "תפרים שטוחים פנימה — לא מגרדים גם אחרי שעות לבישה"
- "כביסה עדינה עד 30°C שומרת על הצבע והרכות"

## Sensory Language — Use These

When describing fabric feel or comfort, use only these approved sensory words:

| Hebrew | Context |
|--------|---------|
| רך | touch / feel |
| נעים | sensation over time |
| קליל | weight / lightness |
| נושם | breathability |
| גמיש | movement / stretch |
| מחבק | fit / warmth |
| עדין | gentleness on skin |
| חלק | surface texture |

**Do not** invent new sensory claims not supported by product data.

---

## No Cross-Section Repetition

Each section must contribute unique information.
If the same phrase appears in two sections — one must be rewritten.

| Section | Its unique angle |
|---------|----------------|
| Hero | emotional hook — what draws the parent in |
| Fabric Story | how the fabric feels on baby's skin |
| Benefits | why each feature helps baby or parent |
| Care | how easy maintenance is |
| FAQ | direct answers to specific doubts |
| Emotional Reassurance | a moment of human connection — not a benefit list |

If two sections say the same thing in different words — that's repetition.
Cut one. Deepen the other.

---

## Care Section — Special Rule

Care instructions must NEVER read as restrictions.

| Restriction phrasing (forbidden) | Advantage phrasing (required) |
|----------------------------------|-------------------------------|
| "ללא אקונומיקה" | "שומר על הסיבים — אין צורך בחומרי הלבנה" |
| "כביסה עדינה בלבד" | "כביסה קלה שומרת על הרכות לאורך זמן" |
| "לא להכניס למייבש" | "ייבוש טבעי מאריך את חיי הבגד" |
| "אסור גיהוץ חזק" | "גיהוץ עדין בחום נמוך — תוצאה נקייה בלי מאמץ" |

Frame every care action as something that **preserves value** — not something the parent must avoid.
