## SECTION 1 — PRODUCT CATEGORY MATRIX

### A. ביגוד תינוקות (Baby Clothing)

#### A1. ביגוד יומי (בגדי גוף, חולצות, מכנסיים)

| פרמטר | ערך |
|---|---|
| **כוונת קונה ראשית** | צורך פרקטי — מידה נכונה, נוחות, עמידות לכביסות |
| **חששות מובילים** | מידה שגויה, בד שמגרד, כיווץ בכביסה, snap-buttons שנשברים |

| **אותות אמון** | טבלת מידות גובה/משקל, הוראות כביסה עם טמפרטורה, ביקורות עם גיל תינוק |
| **נושאי FAQ** | מידה מומלצת, כביסה במכונה, כיווץ, התאמה לעור רגיש |
| **זווית Hero** | קלות + נוחות: "מתלבש בשניות — רך לכל יום" |
| **זווית דחיפות** | "X נמכרו השבוע" (רק אם אמיתי) |
| **זווית מתנה** | לא — אלא אם ערכה/סט |

## SECTION 3 — FALLBACK LOGIC MATRIX

כלל ברזל: **כשמידע חסר — לא ממציאים. מנסחים ניטרלי או מסתירים.**

### Missing Fabric (fabric_type = empty)

| היבט | הנחיה |
|---|---|
| **תוכן מותר** | ניסוח ניטרלי: "בד רך ונעים למגע" |
| **תוכן אסור** | כל אחוז (%), שם בד ספציפי (כותנה/במבוק/פוליאסטר), "אורגני", "היפואלרגני" |
| **ניסוח ניטרלי** | "עשוי מבד רך שנעים לעור התינוק" |
| **סקשנים שמשתנים** | fabric_story → כותרת ניטרלית ללא tags; care → `hand_wash` בלבד; benefits → ללא יתרון בד ספציפי |

### Missing Season (season = empty)

| היבט | הנחיה |
|---|---|
| **תוכן מותר** | "מתאים לכל העונות", יתרונות כלליים |
| **תוכן אסור** | "מושלם לחורף", "נושם לקיץ", כל אזכור עונתי ספציפי |
| **ניסוח ניטרלי** | "מתאים לשימוש יומי לאורך כל השנה" |
| **סקשנים שמשתנים** | hero → ללא אזכור עונה; benefits → ללא יתרון עונתי; urgency → ללא דחיפות עונתית |

### Missing Age Range (target_age = empty)

## SECTION 5 — HERO SYSTEM

### לוגיקת Hero — ביגוד

**נוסחת כותרת:** `[שם מוצר] — [תכונת בד/תועלת עיקרית]`
- דוגמה: "סרבל כותנה פרחוני — רך במיוחד לעור רגיש"
- Fallback (אין fabric): "סרבל פרחוני — נוח וקל ללבישה"

**נוסחת תת-כותרת:** `[עמוד מסרים ראשי] + [פרקטיות]`
- דוגמה: "בד רך שנושם, snap-buttons נוחים, וכביסה קלה"
- Fallback (אין feature): "בד רך ונוח לשימוש יומי"

**נוסחת שורת אמון:** `[משלוח] · [החזרה] · [כוכבים]`
- דוגמה: "משלוח חינם מעל 199₪ · החזרה תוך 14 יום · ⭐ 4.8 (47 ביקורות)"
- Fallback (אין ביקורות): "משלוח חינם מעל 199₪ · החזרה תוך 14 יום · תשלום מאובטח"

**לוגיקת Badges/Chips:**
- `is_new` = true → badge "חדש"
- `gift_suitable` = true → badge "מתאים כמתנה"
- `on_sale` = true → badge "מבצע"
- אף אחד → ללא badge

**CTA:** "הוסיפו לסל" — sticky bar במובייל, כולל מחיר + בחירת מידה.

**דוגמאות גרועות להימנע מהן:**
- ❌ "הבגד הכי טוב לתינוק שלכם" (כוזב + "תינוקכם" blacklisted)
- ❌ "איכות גבוהה במחיר משתלם" (גנרי + "איכות גבוהה" blacklisted)
- ❌ "מושלם לכל תינוק" ("מושלם" blacklisted)

### לוגיקת Hero — הנעלה

**נוסחת כותרת:** `[שם מוצר] — [שלב התפתחותי/תועלת]`
- דוגמה: "נעלי צעד ראשון — סוליה גמישה ואנטי-סליפ"
- Fallback: "נעליים לתינוקות — נוחות ובטוחות"

- דוגמה: "סוליה שלא מחליקה, בד נושם, וקלות לנעילה"

**שורת אמון:** זהה לביגוד.

**דוגמאות גרועות:**
- ❌ "מומלץ ע"י רופאים" (טענה רפואית ללא אסמכתא)
