## PHASE 4 — Doll Type Differences

סטטוס: **פתוח — מחקר חומרים קיים מהדוח, טעון אימות מול מלאי BabyMania**

**מפת חומרים מהדוח (research only — לא מאומת):**
- Full Vinyl → ₪299–₪599 → ילדות 3–6, רחיצה מלאה, עמיד
- Vinyl Head + Cloth Body → ₪299–₪499 → רך לחיבוק, לא לאמבט
- Silicone Head + Cloth Body → ₪799–₪1,190 → "הטוב משני העולמות"
- Full Silicone → ₪1,100+ → קולקציונרים / מבוגרים רגשיים

- [ ] טבלה ללקוח: סיליקון מלא
- [ ] טבלה ללקוח: ראש ויניל/סיליקון + גוף בד
- [ ] מדריך גודל: קטנה / בינונית / גדולה + אינפוגרפיקה עם יד אנושית
- [ ] שיער מושרש מול שיער מצויר — מה ההבדל בפועל

## PHASE 5 — Product Truth Collection

סטטוס: **חסום — ממתין לאישור אייל לפנות לספקית**

**8 שאלות חובה לספקית (מהדוח + תוספות):**
- [ ] חומר כל בובה: ויניל / סיליקון / היברידי?
- [ ] פירוט גודל + משקל לכל בובה
- [ ] פרטי שיער: מושרש / מצויר? האם עמיד?
- [ ] פרטי עיניים: פתוחות / עצומות?
- [ ] מה כלול בקופסה לכל בובה (בגדים, מוצץ, בקבוק, תעודת לידה?)
- [ ] **האם המוצץ מגנטי?** (complaint #1 בשוק — 17% מהביקורות)
- [ ] **האם מידות הבגדים מתאימות לבובה?**
- [ ] תעודות בטיחות: EN71 / ASTM F963 / BPA-free / phthalate-free?
- [ ] מדיניות החזרה בפועל (14 יום? 30 יום? על חשבון מי?)
- [ ] מלאי יציב לאביזרים?
- [ ] האם יש תמונות אמיתיות של הבובות (לא תמונות ספק/סטוק)?

**אחרי קבלת תשובות:**

- [ ] לעדכן סעיף "Product Truth Before Copy" בstate file
